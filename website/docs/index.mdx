# EVA Agent

**EVA Agent** is a complete personal AI agent — **skills, memory, messaging gateway, cron, tools, TUI, and desktop** — under its own brand. This documentation ships with the [Pasqualotty/eva-agent](https://github.com/Pasqualotty/eva-agent) repository.

This tree **is the engine** (MIT fork of [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent)). Hermes is not installed as an external dependency. All engine features below are **EVA features**.

## Install

### From source (canonical EVA repo)

```bash
git clone https://github.com/Pasqualotty/eva-agent.git
cd eva-agent
uv venv --python 3.11
source .venv/bin/activate   # Windows: .\.venv\Scripts\Activate.ps1
uv pip install -e ".[all]"
eva
```

### Platform notes

- **Windows (native):** product home is `%LOCALAPPDATA%\eva`. CLI, gateway, TUI, and tools are intended to work without WSL.
- **Linux / macOS / WSL2:** clone + uv install as above; use `eva` as the product CLI.
- **Termux:** prefer the curated `.[termux]` extra when full extras pull incompatible voice deps.

See the full **[Installation Guide](/getting-started/installation)** for managed installers, layouts, and platform support. Some pages may still say “Hermes” in procedural detail — treat those as **engine docs**; the product brand is **EVA**.

:::tip Fastest path to a working agent
After installing, run `eva setup` (or `eva setup --portal` if using Nous Portal OAuth for model + Tool Gateway). See [Nous Portal](/integrations/nous-portal).
:::

## What is EVA Agent?

It's not a coding copilot tethered to an IDE or a chatbot wrapper around a single API. It's an **autonomous agent** that gets more capable the longer it runs. It lives wherever you put it — a $5 VPS, a GPU cluster, or serverless infrastructure (Daytona, Modal) that costs nearly nothing when idle. Talk to it from Telegram while it works on a cloud VM you never SSH into yourself. It's not tied to your laptop.

## Key Features

- **A closed learning loop** — Agent-curated memory with periodic nudges, autonomous skill creation, skill self-improvement during use, FTS5 cross-session recall with LLM summarization, and [Honcho](https://github.com/plastic-labs/honcho) dialectic user modeling
- **Runs anywhere, not just your laptop** — 6 terminal backends: local, Docker, SSH, Daytona, Singularity, Modal
- **Lives where you do** — CLI, Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Mattermost, Email, SMS, and 20+ platforms from one gateway
- **Scheduled automations** — Built-in cron with delivery to any platform
- **Delegates & parallelizes** — Spawn isolated subagents for parallel workstreams
- **Open standard skills** — Compatible with [agentskills.io](https://agentskills.io)
- **Full web control** — Search, extract, browse, vision, image generation, TTS
- **MCP support** — Connect to any MCP server for extended tool capabilities
- **Research-ready** — Batch processing, trajectory export, RL-oriented tooling
- **Plugins at the edges** — Memory providers, model providers, platforms, context engines
